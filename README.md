# 🌍 TRAVELWORLD - Tour & Travel Booking Application

A full-stack web application built using the MERN stack that allows users to explore various travel destinations, view tour packages, and book their trips seamlessly. 

## ✨ Features

* **Interactive User Interface:** Modern, responsive, and clean design for an optimal user experience.
* **Comprehensive Tour Catalog:** Browse a wide variety of domestic and international tour packages with detailed pricing.
* **Dynamic Booking System:** Real-time price calculation based on user inputs (number of people, dates) with a smooth checkout flow.
* **Personalized Recommendations:** Smart tour suggestions based on user interests (e.g., Nature tours).
* **Secure Authentication:** User registration and login functionality.
* **Review System:** Travelers can share their feedback and experiences.

## 🛠️ Technologies Used

**Frontend:**
* React.js (Vite)
* HTML5, CSS3, JavaScript (ES6)
* React Router for navigation

**Backend:**
* Node.js
* Express.js
* MongoDB (Database)
* JSON Web Tokens (JWT) for authentication
